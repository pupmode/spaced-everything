# Spaced Everything  
  
An Obsidian plugin for spaced note review, based on the concept of  
[spaced everything](https://notes.andymatuschak.org/...) by Andy Matuschak.  
  
## Credits  
  
This plugin is inspired by [spaced-inbox](https://github.com/riceissa/spaced-inbox)  
by [Issa Rice](https://github.com/riceissa).  
  
The "spaced everything" concept is described by Andy Matuschak in his notes on  
[a writing inbox for transient and incomplete notes](https://notes.andymatuschak.org/z5aJUJcSbxuQxzHr2YvaY4cX5TuvLQT7r27Dz).
